Subreddit-Scanner

Scans a series of user given subreddits in python using reddits praw API and displays a GUI to them using PyQt5's API. Allows users to create "multireddits" with subreddits of their choice. They can see these multireddits on their account and if they wish, backup their current subreddits to one, mimic their current reddit subscriptions to match one as well as save posts from them.


Personal Website

With adjustments from HTML5 up's template, along with my knowledge of javascript, HTML and CSS made this website! The particles in the background are made using particles-js's library and are faded in with JQuery commands.


Moodify Me

Currently in development with the end goal of becoming a website where users can message each other. It will distinguish itself by having a mini game built in with points for whoever can guess the other users emotion behind their message. It is currently being developed in Node.js, Flask and Pymongo. In the future it aims to use the data gathered from the mini game to train a machine learning application which can react to users messages based on what emotion the machine believes the message to have.
